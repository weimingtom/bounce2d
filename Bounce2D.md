# Introduction #

## Description ##

Bounce2D is a 2D rigid body physics simulator. It is open source and free for commercial and non comerical use; under the BSD licence.

## Features ##
**2D rigid body physics simulator
Collision detection and reporting
Developed in Java
Open Source**
